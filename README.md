## 📂 项目导航 (Projects Directory)

### 1. [仿真游戏后台数据库架构设计 (MySQL)](./Delta-Action-DB-Architecture)
* **核心亮点:** 运用 Web 检索与大模型辅助构建游戏领域知识库，规模化生成 16 张核心表的 ER 模型与底层 SQL。通过 AI IDE 辅助完成复合索引调优，利用触发器保障高并发下的数据原子性。
* **技术栈:** MySQL, SQL, Database Architecture, Prompt Engineering

### 2. [基于 MapReduce 的社交推荐算法工程级优化](./MapReduce-Social-Recommendation)
* **核心亮点:** 利用大模型进行工程级架构推演，对比“长尾截断”与“Top-K”策略防范数据倾斜导致的 Map 端 OOM。独立搭建 Docker 化的 Hadoop 容器集群。
* **技术栈:** Hadoop, MapReduce, Docker, Distributed Systems

### 3. [AI 驱动的 Web 全栈游戏化平台开发](./Pacman-Web-Fullstack)
* **核心亮点:** 深度运用 Cursor/Trae 等 AI IDE 敏捷构建前后端链路。在“吃豆人”游戏高频状态更新场景下，借助 AI 推演解决多端数据交互冲突，完成实时积分同步与资产持久化。
* **技术栈:** JavaScript, HTML, PHP, MySQL, RESTful API

### 4. [金融交易风控策略分析与异常识别](./Financial-Risk-Analysis)
* **核心亮点:** 牵头千万级交易数据集清洗与分析，基于 Python 完成特征工程，构建 6 个诈骗交易核心行为维度，输出全流程可落地的风控策略闭环。
* **技术栈:** Python, Data Analysis, Feature Engineering, Risk Control
